### `generate types`

```
npm run strapi ts:generate-types
```

### `develop`

```
npm run develop
# or
yarn develop
```

### `start`

```
npm run start
# or
yarn start
```

### `build`

Build your admin panel. [Learn more](https://docs.strapi.io/developer-docs/latest/developer-resources/cli/CLI.html#strapi-build)

```
npm run build
# or
yarn build
```
